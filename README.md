# ParcialAlgo3-

## Descripción

El objetivo del proyecto es practicar el diseño orientado a objetos en Java, utilizando interfaces y herencia para representar la lógica de un imperio intergaláctico y sus estructuras defensivas.

## Características principales

- Modelado de naves espaciales con sistemas de ataque y defensa.
- Implementación de diferentes tipos de armamento y escudos.
- Uso de interfaces para abstraer comportamientos de ataque y defensa.
- Proyecto configurado con Maven y compatible con Java 11.

## Estructura del proyecto

- `src/main/java/ImperioIntergalactico/`: clases del dominio espacial (naves, ataques y defensas).
- `src/main/java/org/example/Main.java`: clase principal con un ejemplo básico de ejecución.
- `src/test/java/pruebas.java`: archivo de pruebas inicial.
- `pom.xml`: configuración de compilación y dependencias.

## Tecnologías

- Java 11
- Maven
- JUnit y TestNG (dependencias para pruebas)

## Uso

1. Clonar el repositorio.
2. Ejecutar `mvn compile` para compilar el proyecto.
3. Ejecutar `mvn test` para correr pruebas.
4. Ejecutar la clase `org.example.Main` para ver la salida de ejemplo.

## Descripción para portafolio

Este proyecto demuestra conocimientos en diseño orientado a objetos y construcción de aplicaciones Java mediante Maven. La implementación utiliza patrones básicos de abstracción y separación de responsabilidades para gestionar sistemas de ataque y defensa en una simulación de naves espaciales.
